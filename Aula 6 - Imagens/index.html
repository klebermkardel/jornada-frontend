Imagens

Usamos para mostrar imagens em nossas telas para os usuários.

Representado pela tag img.

A tag img não faz nada sozinha, precisamos do atributo src para renderizar/mostrar aos usuários.

Exemplo:

< img src="imagem.jpg" >
< img src="imagem.png" >
< img src="imagem.svg" >

Uma imagem pode ser renderizada em vários formatos como:

  • JPEG
  • GIF (animados)
  • PNG
  • APNG
  • SVG
  • BMP
  • BMP ICO
  • PNG ICO

Preste atenção sempre no formato das suas imagens, pois pode ter problema na hora de renderizar para o usuário final.

Para completar, vamos usar o atributo alt.

Ele é necessário para o uso de imagem.

Exemplo: < img src="minha-imagem.jpg alt="Minha imagem >

Caso a imagem tenha algum problema para carregar, aparecerá um texto para orientar o usuário sobre o que se trata a imagem e também serve para acessibilidade a usuários com deficiência visual que utilizam softwares leitores de tela.
