Tabelas

Usamos quando se tem um conjunto de dados que precisam ser apresentados ordenadamente

Representada pela tag table

Nunca utilizamos apenas a tag table, junto a ela temos outras como:

< table > -> tabela / atributo border="1" cria bordas na tabela

< tr > -> linha da tabela

< td > -> coluna

< /tr > -> fim da linha

< /table > -> fim da tabela

Dica de ouro: uma tabela tem mais atributos, eles são:

  • thead: separa o cabeçalho da tabela
  • tbody: separa o corpo
  • tfoot: separa o final

Veja abaixo um exemplo de tabela:

# Primeiro Segundo
1 João Mariano
2 Márcio Rafael
# Primeiro Segundo
