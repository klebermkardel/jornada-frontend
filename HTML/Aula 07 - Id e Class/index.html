Id e Class

São uma forma de identificar elementos em seu HTML.

É muito importante saber usá-los pois são muito utilizados em todos os códigos por aí.

Vamos utilizá-los com CSS ou JavaScript.

Tudo depende da sua utilização.

Cada id tem um nome próprio por página. Isso mesmo, apenas 1 nome de id por página.

Se uma página tiver mais de 1 id com nome igual está incorreto.

Exemplo:

Página 1 - Forma Errada

< h1 id="meu-primeiro-id" >Conteúdo< /h1 >

< h2 id="meu-primeiro-id" >Conteúdo< /h2 >

Página 2 - Forma Correta

< h1 id="meu-primeiro-id" >Conteúdo< /h1 >

< h2 id="meu-segundo-id" >Conteúdo< /h2 >

Pode ser inserido qualquer nome neles.

Já com a class conseguimos utilizar mais de 1 vez por página.

Identificamos um grupo de elementos, reaproveitando o código deixando mais ágil nosso desenvolvimento.

Cuidado para não utilizá-los de uma forma errada.

Dependendo de como for usado podemos gerar códigos desnecessários poluindo e dificultando a manutenção do nosso código.
