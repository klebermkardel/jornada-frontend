O que são tags?

São etiquetas usadas para informar ao navegador como deve ser apresentado o site.

Quando escrevemos tags no HTML elas serão interpretadas pelo navegador.

Como representar:

Tags são representadas com o sinal < code > representando a abertura da tag e < /code > representando o seu fechamento.

Cada tag tem um propósito, como: links, parágrafos, listagens, etc..
