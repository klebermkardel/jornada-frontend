Formulários

Usados para enviar informações para uma determinada funcionalidade do sistema.

Com eles temos várias possibilidades de comunicação.

Podemos enviar informações para o Back End, submeter dados e salvar no localStorage, entre outros.

Representado pela tag form

Nunca usamos apenas a tag form, junto a ela temos outras como:

  • input
  • textarea
  • button
  • select
  • fieldset
  • label
  • ...

Formas de uso:

< form > -> formulário com tipo de envio
< label for="nome" >Nome:< /label > -> Título
< input type="text" id="nome" > -> Campo
< /form > -> fim do formulário
