O que são divs?

O elemento divisão HTML div é um container genérico para conteúdo.

É usado sempre quando vamos montar o layout em elementos genéricos em que não são usados a síntaxe

Exemplo do código:

< div id="listagem-principal" class="container" >
  < ul >
    < li >Listagem 1< /li >
    < li >Listagem 2< /li >
    < li >Listagem 3< /li >
    < li >Listagem 4< /li >
  < /ul >
< /div >
