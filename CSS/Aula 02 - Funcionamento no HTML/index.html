CSS - Funcionamento no HTML

Externo, interno ou inline?

Existem 3 maneiras principais de adicionar código CSS às suas páginas HTML.

O código CSS (ou folha de estilo) pode ser externo, interno ou inline.

Externo: a maneira mais conveniente de adicionar CSS ao seu site, é vinculá-lo a um arquivo .css externo.

Dessa forma, quaisquer alterações feitas em um arquivo CSS externo serão refletidas em seu site globalmente.

Interno: é colocado na tag head de uma determinada página.

As Classes e IDs podem ser usadas para se referir ao código CSS, mas eles só serão ativos nessa página específica.

Inline: é usado para uma tag HTML específica.

O atributo "style" é usado para formatar uma tag HTML específica.

Não é reconhecido, pois cada tag HTML precisa ser denominada individualmente.
