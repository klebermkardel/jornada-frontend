No exemplo acima foram criados um article e dentro dele duas sections com a classe container e em cada section um parágrafo com id texto-1 e texto-2 respecticamente para que possamos identificar como são utilizados os principais seletores utilizados no CSS que são nome da tag, class e id.

Definimos através do seletor "nome da tag" que o article teria uma largura de 200px e margin 0 auto para centralizar o conteúdo horizontalmente na tela. E definimos também que as sections com class ",container" teriam um background vermelho e por último foram definidos que o parágrafo de id "#texto-1" teria a cor de texto branca e o de id "texto-2 uma cor um pouco mais escura com padding de 30px".

Dessa forma pudemos observar como funcionam os seletores e como devemos utilizá-los ao decorrer do código.
