CSS - Introdução

O CSS significa Cascading Style Sheets e refere-se a um conjunto de estilos que podem ser inseridos e configurados na edição de uma página de web.

Resumindo o CSS faz a magia e da vida ao nosso querido HTML.

O CSS é uma linguagem de web muito importante para o desenvolvimento de sites e aplicaçõs que utilizam HTML.

Ele faz parte da tríada da web ao lado do JavaScript e do HTML.

Como ele funciona?

Se você codar e optar para que o parágrafo apareça com o fundo rosa e com o texto em negrito para as pessoas que visualizam sua página utilizando CSS, ele transformará no efeito pretendido.

Veja abaixo um exemplo de código HTML com o CSS aplicado para que o fundo fique rosa e a cor do texto fique branca e em negrito e tenha um espaçamento ao redor do texto de 30 píxeis:

< div class="box" >
  Texto
< /div >

O código acima não representa nada, apenas o código HTML, mas sem nenhuma aplicação do CSS.

Para que esse código adquira as propriedades CSS passadas a ele devemos abrir a tag style dentro da tag head e escrever o código CSS desejado

Exemplo:

< style >
  .box {
     background: pink;
     color: #fff;
     font-weight: bold;
     padding: 30px;
  }
< /style >
